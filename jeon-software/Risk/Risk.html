Instructions

Please select from among five different gambles the one gamble you would like to play. The five different gambles are listed on your GAMBLE SELECTION TABLE. You must select only one of these gambles.

Each gamble has two possible outcomes (Event A or Event B) with the indicated probabilities of occurring. Your earnings will be determined by: 1) which of the five gambles you select; and 2) which of the two possible events occur. For every gamble, each event has a 50% chance of occurring.

At the end of the study, if this part is selected for payment, you will make a random selection between Event A and Event B.

For example: If you select gamble 4 and Event A occurs, you will earn $40. If Event B occurs, you will earn $4.
